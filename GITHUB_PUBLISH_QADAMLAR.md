## üöÄ Keyingi Qadam: GitHub ga Publish Qilish

### Qadam 1: Publish Repository

1. GitHub Desktop da **yuqorida o'ng tomonda** "Publish repository" bo'limini toping
2. **"Publish your repository to GitHub"** tugmasini bosing (yoki "Publish repository" tugmasini)

### Qadam 2: Repository Sozlamalari

1. Quyidagilarni to'ldiring:
   - **Name:** `Futbot` (yoki istalgan nom)
   - **Description:** `Stadium booking bot` (ixtiyoriy)
   - **"Keep this code private"** ni o'chirib qo'ying (yoki qoldiring)
2. **"Publish repository"** tugmasini bosing

### Qadam 3: Tekshirish

1. Bir necha soniyadan keyin repository GitHub ga yuklanadi
2. GitHub Desktop da **"View on GitHub"** tugmasi ko'rinadi
3. Tugmani bosib, GitHub da repository ni ko'ring
