## üîç Muammo: Kod yuklanmayapti

Agar kod yuklanmayotgan bo'lsa, quyidagi usullarni sinab ko'ring:

---

## ‚úÖ Usul 1: GitHub Desktop - Yangi Repository Yaratish

### Qadam 1: Yangi Repository
1. **GitHub Desktop** ni oching
2. **"File"** ‚Üí **"New Repository"** (yoki Ctrl+N)
3. Quyidagilarni to'ldiring:
   - **Name:** `Futbot`
   - **Description:** `Stadium booking bot`
   - **Local path:** `C:\Users\Acer\Desktop\fut bto`
     - Yoki **"Choose..."** ni bosib, papkani tanlang
   - **"Initialize this repository with a README"** ni **O'CHIRING** (bo'sh qoldiring)
   - **Git ignore:** `Node` ni tanlang
   - **License:** `None` (yoki istalgan)
4. **"Create Repository"** ni bosing

### Qadam 2: Barcha Fayllarni Ko'rish
1. GitHub Desktop da barcha fayllar ko'rinishi kerak
2. Agar ko'rinmasa, **"Show in Explorer"** ni bosing va fayllarni ko'ring

### Qadam 3: Commit Qilish
1. Pastda **"Commit to main"** bo'limida:
   - **Summary:** `Stadium booking bot - complete project`
   - **Description:** (ixtiyoriy)
2. **"Commit to main"** tugmasini bosing

### Qadam 4: GitHub ga Yuklash
1. Yuqorida **"Publish repository"** tugmasini bosing
2. Quyidagilarni to'ldiring:
   - **Name:** `Futbot`
   - **Description:** `Stadium booking bot`
   - **"Keep this code private"** ni o'chirib qo'ying (yoki qoldiring)
3. **"Publish repository"** tugmasini bosing

## ‚úÖ Usul 2: GitHub Desktop - Mavjud Papkani Qo'shish

### Qadam 1: Add Local Repository
1. **GitHub Desktop** ni oching
2. **"File"** ‚Üí **"Add Local Repository"**
3. **"Choose..."** ni bosing
4. **File Explorer** da:
   - Address bar ga: `C:\Users\Acer\Desktop\fut bto` yozing
   - Yoki Desktop ‚Üí "fut bto" papkasini tanlang
5. **"Add"** ni bosing

### Qadam 2: Agar "This directory does not appear to be a Git repository" chiqsa:
1. **"create a repository"** ni bosing
2. **"Create Repository"** ni bosing
3. Keyin **Qadam 3** ga o'ting

### Qadam 3: Commit va Publish
1. Barcha fayllarni tanlang
2. **"Commit to main"** ni bosing
3. **"Publish repository"** ni bosing

## ‚úÖ Usul 3: GitHub Web Interface orqali

Agar GitHub Desktop ishlamasa:

1. [GitHub.com](https://github.com) ga kiring
2. **"New repository"** ni bosing
3. Repository nomi: `Futbot`
4. **"Create repository"** ni bosing
5. Keyin **GitHub Desktop** da:
   - **"File"** ‚Üí **"Clone Repository"**
   - Repository ni tanlang
   - Local path: `C:\Users\Acer\Desktop\fut bto` (yoki yangi papka)
   - **"Clone"** ni bosing
6. Barcha fayllarni ko'chiring
7. Commit va Push qiling

## üîß Muammolar va Yechimlar

### Muammo 1: "This directory does not appear to be a Git repository"
**Yechim:** "create a repository" ni bosing

### Muammo 2: Papka ko'rinmayapti
**Yechim:** 
- Address bar ga to'g'ridan-to'g'ri yo'lni yozing: `C:\Users\Acer\Desktop\fut bto`
- Yoki Desktop ‚Üí "fut bto" ni tanlang

### Muammo 3: Fayllar ko'rinmayapti
**Yechim:**
- `.gitignore` faylini tekshiring
- Barcha fayllar qo'shilganligini tekshiring

### Muammo 4: "Publish repository" tugmasi ko'rinmayapti
**Yechim:**
- Avval **"Commit to main"** ni bosing
- Keyin "Publish repository" ko'rinadi

### Muammo 5: Push xatosi
**Yechim:**
- GitHub akkauntingiz bilan to'g'ri kirganligingizni tekshiring
- Internet aloqasini tekshiring
- Repository nomi to'g'ri ekanligini tekshiring
