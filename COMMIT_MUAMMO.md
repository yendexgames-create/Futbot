# "Commit to main" Tugmasi Ko'rinmayapti - Yechim

## üîç Muammo: "Commit to main" tugmasi ko'rinmayapti yoki ishlamayapti

### ‚úÖ Yechim 1: Barcha Fayllarni Tanlash

1. **GitHub Desktop** da chap tomonda barcha fayllar ko'rinishi kerak
2. Agar fayllar ko'rinmasa:
   - **"Show in Explorer"** ni bosing
   - Fayllar mavjudligini tekshiring
3. **Barcha fayllarni tanlang** (checkbox larni belgilang)
4. Keyin "Commit to main" ko'rinishi kerak

### ‚úÖ Yechim 2: Yangi Repository Yaratish

Agar hali ham ishlamasa, yangi repository yarating:

1. **GitHub Desktop** da:
   - **"File"** ‚Üí **"New Repository"** (Ctrl+N)
2. Quyidagilarni to'ldiring:
   - **Name:** `Futbot`
   - **Local path:** `C:\Users\Acer\Desktop\fut bto`
   - **"Initialize this repository with a README"** ni **O'CHIRING**
   - **Git ignore:** `Node` ni tanlang
3. **"Create Repository"** ni bosing
4. Barcha fayllar ko'rinishi kerak
5. Pastda **"Commit to main"** bo'limida:
   - **Summary:** `Stadium booking bot`
   - **"Commit to main"** ni bosing

### ‚úÖ Yechim 3: Mavjud Repository ni O'chirib, Qayta Yaratish

1. **GitHub Desktop** da:
   - **"File"** ‚Üí **"Options"** ‚Üí **"Repositories"**
   - Mavjud repository ni o'chiring
2. Keyin **Yechim 2** ni bajaring

### ‚úÖ Yechim 4: To'g'ridan-to'g'ri Publish

Agar fayllar allaqachon commit qilingan bo'lsa:

1. **GitHub Desktop** da yuqorida **"Publish repository"** tugmasini qidiring
2. Agar ko'rinmasa:
   - **"View"** ‚Üí **"Show Changes"** ni bosing
   - Yoki **"Repository"** ‚Üí **"Push"** ni bosing
